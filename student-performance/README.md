# 🎓 Student Performance Predictor

## 🚀 Overview

This project predicts student final grades using Machine Learning.

It is a complete end-to-end ML application with:

* Data Analysis (EDA)
* Model Training
* Flask API
* Web Interface

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Flask
* HTML

---

## ⚙️ Features

* Predict student performance based on inputs
* Interactive web interface
* Real-time predictions

---

## 📊 Model

* Linear Regression
* Random Forest Regressor

---

## 🚀 How to Run

```bash
cd app
python app.py
```

Then open:

```
http://127.0.0.1:5000/
```
